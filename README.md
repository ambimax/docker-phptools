 - codebeautifier2 
 - codebeautifier3
 - codecept
 - codesniffer2
 - codesniffer3
 - composer
 - phpmd
 - phpunit5
 - phpunit6
 - phpunit7
 - mhsendmail
 - modman
 - n98-magerun
 - zettr

### Location

All tools are save to `/tools`
